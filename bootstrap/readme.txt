squareBracket Bootstrap Theme

Our custom Bootstrap theme for squareBracket. Originally in the Milestone 1 days it was some Bootstrap 3 edit, but we updated to Bootstrap 5 in May 2021.

The Dashboard and Signin css files are from the Bootstrap 5 documentations.

This will be deprecated once sbnext replaces the Bootstrap layout.

Animations are disabled thanks to button hovering, this is low-prioity thanks to sbnext.
